App Weather

weather es una aplicación web desarrollada para monitoriar datos del clima, utilizando microcontroladores y sensores, la app se desarrolló con el framework de Flask para Python, Chart.js para graficar los datos en tiempo real. para el alojamiento de los datos se utilizó Mariadb, bootstrap 5 para los estilos y el diseño responsive, y por ultimo el despliegue se realizó en los servicios de aws .
